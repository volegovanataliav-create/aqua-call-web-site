# Начало работы

Добро пожаловать в AquaCall! Это руководство поможет вам быстро начать работу с нашим сервисом.

## Установка

Для начала работы с AquaCall выполните следующие шаги:

### 1. Клонирование репозитория

```bash
git clone https://github.com/your-org/aqua-call-web-site.git
cd aqua-call-web-site
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск локального сервера

```bash
npm run docs:dev
```

Сайт будет доступен по адресу `http://localhost:5173`

## Структура проекта

```
aqua-call-web-site/

```

## Основные команды

### Разработка

```bash
npm run docs:dev
```

Запускает локальный сервер разработки с hot-reload.

### Сборка

```bash
npm run docs:build
```

Создает production сборку сайта в директории `.vitepress/dist`.

### Превью

```bash
npm run docs:preview
```

Предварительный просмотр production сборки локально.

## Технологии

AquaCall использует следующие технологии:

- **VitePress** - генератор статических сайтов на основе Vite
- **Vue 3** - прогрессивный JavaScript фреймворк
- **Tailwind CSS** - utility-first CSS фреймворк
- **TypeScript** - типизированный JavaScript
